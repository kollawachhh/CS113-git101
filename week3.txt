Control Flow

    การใช้ Loop

    1. if - else เป็นการกำหนดเงื่อนไข
        if (expression) >> เงื่อนไข
            statement1 >> เมื่อผ่านเงื่อนไขจะมาทำใน statement1
        else >> ถ้าไม่เข้าเงื่อนไขใน block if จะมาอ่านที่ else จากนั้นทำ statement2
            statement2 
        คำสั่ง else if จะเข้าคำสั่งใดคำสั่งหนึ่ง ถ้ามีหลาย else if
        คำสั่ง if ถ้ามีหลายๆอัน จะเช็คทีละ if
    
    2. while เป็นการวนจนกว่าจะไม่เข้าเงื่อนไข
        while (expression)  >> เช็คเงื่อนไขถ้าถูกก้จะทำไปเรื่อยๆ ถ้าไม่เข้าเงื่อนไขก้จะไม่ทำ
            statement >> ทำเมื่อเข้าเงื่อนไข

    3. For คล้ายๆกับ while แต่จะสามารถกำหนดรอบได้และเขียนได้สั้นกว่า
        for (expr1; expr2; expr3) >> expr1 คือกำหนดตัวแปรเริ่มต้น expr2 คือเงื่อนไขเข้าลูป expr3 คือการเพื่มค่าของตัวแปรที่เช็คเงื่อนไข
            statement >> ทำเมื่อเข้าเงื่อนไข

    4. do - while  เป็นลูปที่จะทำใน block คำสั่งของ do ก่อน
        do >> ทำในคำblock นี้ก่อน
            statement 
        while (expression); >> ทำเมื่อ block do เสร็จแล้ว

    5. switch - case เป็นลูปที่เช็คเงื่อนไขเป็นเคสๆ คล้ายๆคำสั่ง if
        switch (expression) { >> ตัวแปรหรือค่าที่จะมาเปรียบเทียบเงื่อนไขแต่ละ case
        case const-expr: >> ทำหน้าที่เช็คเงื่อนไข
            statements >> ทำเมื่อผ่านเงื่อนไขของ case
        default: >> ทำงานเมื่อเงื่อนไขไม่ตรงกับ case ใดเลย
            statements 
        }

    6. คำสั่ง break และ continue
        break >> จะหยุดลูปทันทีเมื่อเจอคำสั่งนี้
        continue >> จะบังคับการทำงานของลูปให้เกิดการวนซ้ำขึ้นมาใหม่ โดยที่ ในกรณีของ while กับ do-while การทำงานจะกระโดดไปที่การตรวจสอบเงื่อนไข (expression) ส่วนในกรณีของ for นั้น การทำงานจะกระโดดไปที่ expr3 แล้วถึงไปตรวจสอบเงื่อนไข (expr2)
